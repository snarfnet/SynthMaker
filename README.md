# SynthMaker

TB-303風のアシッドベース・シンセアプリです。暗い機材パネル、アンバーLED、シアンの信号線を軸にしたUIへ更新しています。

## 主な画面

- オシレーターのチューニングと波形切り替え
- カットオフ、レゾナンス、エンベロープ量の操作
- ADSRエンベロープ
- ベース鍵盤
- 8ステップのパターンパッド
- 生成画像を使ったシンセ・パネル背景

## 設定

- Bundle ID: `com.tokyonasu.SynthMaker`
- Team ID: `83VGKGSQUH`
- Minimum iOS: 15.0
- Audio: AVAudioEngine / AVAudioUnitSampler / AVAudioUnitEQ
